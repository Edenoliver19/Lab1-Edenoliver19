# Lab1: Grade Generator and Organizer Scripts
**Name:** Eden Oliver

## Project Description
This project includes a Python script `grade-generator.py` that calculates weighted grades, GPA, and pass/fail status, and exports assignment data to a CSV file. 
It also includes a Bash script `organizer.sh` that archives CSV files with a timestamp and logs the action.

## How to Run

1. Run Python script:
    python3 grade-generator.py

2. Run Organizer script:
    ./organizer.sh

## Output
- CSV files are created and then archived.
- Logs are saved in `organizer.log`.
